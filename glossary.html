Ingles / English Español / Spanish Definición / Definition
Artificial Intelligence Inteligencia Artificial Tecnología que simula la inteligencia humana. / Technology that simulates human intelligence.
Interface Interfaz de Usuario Permite interactuar y realizar acciones de manera clara y ordenada. / It allows you to interact and perform actions in a clear and orderly manner.
Figma Figma Permite diseñar interfaces y prototipos de forma colaborativa en línea. / It allows you to design interfaces and prototypes collaboratively online.
Game UI Interfaz de Usuario de Videojuegos Elementos visuales y controles que permiten al jugador interactuar con el juego. / Visual elements and controls that allow the player to interact with the game.
UX Design Diseño de Experiencia de Usuario Crear productos fáciles, útiles y agradables para quienes los usan. / Create products that are easy, useful, and enjoyable for their users.
Game Design Diseño de Videojuegos Crear las reglas y experiencias que definen cómo se juega un videojuego. / Create the rules and experiences that define how a video game is played.
Prototyping Prototipado Crear una versión preliminar para probar ideas. / Create a preliminary version to test ideas.
Player Experience Experiencia del Jugador Cómo el jugador vive y siente el juego. / How ​​the player experiences and feels the game.
Visual Design Diseño Visual Comunicar ideas de forma clara y atractiva mediante elementos gráficos. / Communicate ideas clearly and attractively using graphic elements.
Interactive Systems Sistemas Interactivos Responde a las acciones del usuario y permite una comunicación dinámica. / It responds to user actions and enables dynamic communication.
Generative AI IA Generativa Crear contenido nuevo a partir de datos. / Create new content from data.
Automation Automatización Realizar tareas sin intervención humana. / Perform tasks without human intervention.
Future of Work Futuro del Trabajo Cómo cambian las formas de trabajar con la tecnología. / How ​​technology is changing the way we work.
Digital Transformation Transformación Digital Uso de tecnología para mejorar y modernizar procesos en una organización. / Use of technology to improve and modernize processes in an organization.
Data Analysis Análisis de Datos Analizar datos para obtener información útil. / Analyze data to obtain useful information.
Technology Tecnología Uso de herramientas y conocimientos para crear soluciones. / Using tools and knowledge to create solutions.
Innovation Innovación Capacidad de crear o mejorar algo para generar valor. / Ability to create or improve something to generate value.
Software Development Desarrollo de Software Proceso de crear aplicaciones mediante código y herramientas digitales. / The process of creating applications using code and digital tools.
Machines Maquinas Una máquina es un dispositivo que transforma energía para realizar una tarea o resolver un problema. / A machine is a device that transforms energy to perform a task or solve a problem.
Robot Robot Maquina programable capaz de desarrollar tareas de forma automática o controlada. / Programmable machine capable of performing tasks automatically or in a controlled manner.
Accessibility Accesibilidad Práctica de diseñar y desarrollar aplicaciones, sitios web o software para que puedan ser utilizados por todas las personas / The practice of designing and developing applications, websites, or software so that they can be used by everyone.
Algorithm Algoritmo Es un conjunto ordenado y finito de pasos o instrucciones que se siguen. / It is an ordered and finite set of steps or instructions that are followed.
Usability Usabilidad Es la facilidad con la que los usuarios pueden aprender, entender y utilizar una aplicación, sitio web. / It is the ease with which users can learn, understand, and use an application or website.
Responsive Design Diseño Adaptativo Técnica de desarrollo web que permite que una aplicación o sitio web se ajuste a diferentes tamaños de pantalla y dispositivos / A web development technique that allows an application or website to adapt to different screen sizes and devices.
Animation Animación Efecto visual que crea movimiento o cambios graduales en los elementos de una interfaz, / Visual effect that creates movement or gradual changes in the elements of an interface.
Game Engine Motor de Videojuegos Software que proporciona las herramientas necesarias para crear videojuegos como gráficos, físicas, sonido, animaciones y control de eventos. / Software that provides the necessary tools to create video games such as graphics, physics, sound, animations, and event control.
Deployment Despliegue Proceso de publicar e instalar una aplicación o sistema en un entorno donde los usuarios puedan acceder. / Process of publishing and installing an application or system in an environment where users can access it.
Pixel Density Densidad de Pixeles Cantidad de píxeles que hay en una determinada área de una pantalla. / Number of pixels in a given area of ​​a screen
General AI IA General Capaz de razonar y aprender como un humano en cualquier tipo de tarea. / Able to reason and learn like a human in any type of task.
Narrow AI IA Estrecha Diseñada para tareas específicas. / Designed for specific tasks.
