Conference / Conferencia 1: Game UI Systems / Sistemas de interfaz de usuario de juegos

Speaker / Conferencista

Andrés Felipe Pisso Tobar

Tema principal

Como pasar de pantallas bonitas a interfaces que funcionan

Main topic

How to move from pretty screens to interfaces that actually work

Resumen en Español

La conferencia explicó cómo paso a paso podemos diseñar las interfaces gráficas de los videojuegos mediante herramientas como Figma. El conferensita dio como demostración la importancia de poder crear componentes que sean reutilizables, menús organizados, barras y botones interactivos para mejorar la experiencia del usuario.

Summary in English

The conference explained step by step how we can design video game graphical interfaces using tools like Figma. The speaker demonstrated the importance of creating reusable components, organized menus, and interactive bars and buttons to improve the user experience.
